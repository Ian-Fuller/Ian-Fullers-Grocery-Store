Change Database Permissions

To edit the database, the user needs write permissons, these can be changed by:

- Going to "C:\ProgramData\Ian's Grocery Store"

- Right click on "StoreDatabase.mdf" and select "Properties"

- Go to the "Security" tab, and click "Edit..."

- Select "Users" under "Group or user names:", then check "Full Control" under the "Allow" Column. Click "Apply"

- Repeat steps 2-4 with "StoreDatabase_log.ldf"

- The program will need to be restarted for the changes to go into affect
