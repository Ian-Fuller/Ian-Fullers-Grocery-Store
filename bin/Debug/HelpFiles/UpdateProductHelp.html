Update Product

Select the product you want to update in the first combo box, then select the column you want to update in the second combo box. Type the new value in the textbox and click the \"Update\" button to update the product. If you choose to change the products image, a button will show up for you to select an image (preferably a 100x100 image). If you change the UnitsInStock column, a report will be generated that will show all of the manager purchases.
